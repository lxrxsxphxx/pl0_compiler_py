# PL/0 Compiler

Entwicklung eines Compilers für die Programmiersprache PL/0 in Python

## Projektbeschreibung
<!-- TODO -->
Der Compiler wird in Python implementiert und basiert auf einem Top-Down-Parser (rekursiver Abstieg) ohne Nutzung von Scanner- oder Parsergeneratoren.

## Rahmenbedingungen

- Programmiersprache: Python
- Parser: Top-Down (rekursiver Abstieg)
- Scanner-/Parser-Generatoren: keine
- Erweiterung: Ausgabe von Zeichenketten mit ! (4)

## Funktionsumfang
<!-- TODO -->
- Compiler funktioniert vollständig,
- PL/0-Programme werden kompiliert
